# Project

Trellis is rogueoak's shared AI-agent convention framework: the structure that helps every
project grow. It distributes one opinionated set of rules (and, later, shared templates) that
any rogueoak repo can install and keep in sync, so every repo behaves the same way instead of
each reinventing its conventions.

It is distributed like Spectra - a marketplace plugin for Claude Code, OpenAI Codex, Gemini CLI,
and Cursor - installed with `/trellis-install` and updated with `/trellis-update`. Trellis itself
is developed under Spectra and follows its own rules (it dogfoods itself).
