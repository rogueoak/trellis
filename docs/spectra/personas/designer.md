# 🎨 Designer

See `persona.md` for how to review and comment.

Review whether the change looks and feels consistent, and is clear to the end user.

Check:
- **Consistent sizing** — buttons, inputs, and controls match the established sizes; no one-off
  dimensions for an element that already has a standard.
- **Consistent spacing** — margins, padding, and gaps follow the existing rhythm; no ad-hoc
  pixel nudges where a spacing scale exists.
- **Design tokens over custom styles** — reuse the palette/typography/spacing tokens; flag
  hardcoded colors, font sizes, or magic numbers that duplicate an existing token.
- **Clear calls to action** — one primary action per view; no competing or duplicate CTAs;
  button/link text says what it does (no vague or misleading labels).
- **Considered states** — hover/focus/disabled, empty, loading, and error states look
  intentional, not afterthoughts; layout holds up responsively.
